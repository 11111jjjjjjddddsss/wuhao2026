# 协议与隐私 Runbook

最后更新：2026-06-05

## 当前状态

设置页“服务协议”当前是本地内置目录页，点开后包含 6 个二级页面：

- 用户协议
- 隐私政策
- 第三方信息共享清单
- 个人信息收集清单
- 应用权限
- 风险提示

这些页面都在 Android 本地渲染，不依赖联网加载，也不跳转外部网页。

2026-06-01 已按手机号账号和阿里云融合认证 SDK 复核 6 个页面：隐私政策、第三方信息共享清单、个人信息收集清单和应用权限已加入手机号认证、短信验证码、本机号码一键登录、电话状态、网络状态 / Wi-Fi 状态、登录账号标识、认证 token 和脱敏手机号说明；本版本不提供真实支付、自动续费、礼品卡兑换或农资商品交易，但不把未来业务写死为永不开展；也不虚构当前已经接入支付 SDK、礼品卡兑换、农资商品交易或 SLS 业务日志；Redis 仅用于手机号认证、主聊天用户级频控、App 自动日志接收、帮助与反馈用户发消息和上传短期限流，不在 App 文案里单独包装成业务功能。

2026-06-05 复核 debug / release merged manifest：阿里云融合认证 SDK 和 AndroidX WorkManager 会合并网络变更、用户解锁、唤醒、开机后调度和前台服务相关权限。App 内“隐私政策 / 应用权限”已补充后台待发送任务口径：这些权限只用于带图消息在 App 切后台、进程被系统回收或设备重启后按同一条待发送任务有限重试，不用于 App 外推送、广告通知、定位、通讯录或短信读取。

## 当前权限真相

当前 `AndroidManifest.xml` 及引入的阿里云融合认证 SDK 合并后会声明：

- `INTERNET`
- `ACCESS_NETWORK_STATE`
- `ACCESS_WIFI_STATE`
- `CHANGE_NETWORK_STATE`
- `CHANGE_WIFI_STATE`
- `READ_PHONE_STATE`
- `RECEIVE_USER_PRESENT`
- `WAKE_LOCK`
- `RECEIVE_BOOT_COMPLETED`
- `FOREGROUND_SERVICE`
- `REQUEST_INSTALL_PACKAGES`

后台任务权限当前口径：

- `WAKE_LOCK`、`RECEIVE_BOOT_COMPLETED` 和 `FOREGROUND_SERVICE` 来自 AndroidX WorkManager 相关依赖合并，用于待发送图文消息在后台、进程恢复或设备重启后有限重试。
- 这些权限不上传聊天正文、AI 回复全文、图片内容、图片 URL、手机号、token 或模型 Key；实际日志仍走 App 自动日志脱敏规则。
- 当前不做 App 外推送通知，不包含广告通知或营销触达；若后续真的引入通知、统计、推送或更长后台运行，必须重新复核权限页、隐私政策和第三方清单。

当前不申请：

- 精确定位 / 粗略定位权限
- App 相机权限
- 相册 / 存储读写权限
- 录音权限
- 通讯录权限
- 短信权限
- App 外通知 / 推送权限

图片入口当前口径：

- 照片入口使用 Android 系统 Photo Picker，只访问用户本次主动选择的图片。
- 拍照入口调用外部相机，通过 FileProvider 临时授权写入 App 创建的 URI，不申请 App 相机权限。
- Android Q+ 拍照成功后会把原始照片另存到系统相册 `Pictures/农技千查`，便于用户找回现场照片；输入框、上传和历史展示仍使用 App 私有副本，不额外申请存储权限。

## 当前文案口径

- App 正文不暴露具体模型品牌、模型平台名或供应商账号信息。
- 用户协议和隐私政策使用“第三方大模型和云服务”这类必要委托处理口径。
- 第三方信息共享清单只列当前必要类别；当前使用 ECS / RDS / OSS / 第三方大模型服务，账号登录主线会使用阿里云融合认证 SDK、手机号认证和短信验证码服务；SLS 业务日志、支付 SDK 等未接入业务主链的能力不能写成已接入事实；Redis 只用于手机号认证、主聊天用户级频控、App 自动日志接收、帮助与反馈用户发消息和上传短期限流，不能写成聊天内容、会员、订单或长期资产主链。
- 本版本不包含广告、地图、推送、统计 SDK、支付 SDK、友盟、Bugly、极光或 Firebase；一键登录只按阿里云融合认证 SDK 当前真实接入能力描述，未完成真机回归前不写成已公开稳定上线。
- 本版本不提供真实支付、自动续费、礼品卡兑换或农资商品交易入口，不产生真实扣费、兑换权益或商品订单；服务范围变化时，必须在使用前明确价格、商品或服务内容、支付方式、退款 / 售后规则、权益生效方式和必要资质信息。
- 当前“删除所有历史对话”只删除问诊聊天历史、A/B/C 记忆和 30 天归档，不等于完整账号注销或全部个人信息删除。

- 是否新增个人信息类型。

- 是否新增 Android 权限。
- 是否改变图片、聊天、反馈、日志或支付数据的保存期限。
- 是否改变删除历史对话、注销账号或数据删除范围。
