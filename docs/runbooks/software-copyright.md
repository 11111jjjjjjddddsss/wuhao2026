# 软件著作权登记材料入口

最后更新：2026-05-28

## 当前结论

- 软件著作权登记和 ICP / App 备案可以并行，不需要等备案通过后再启动软著。
- 软著登记的是当前 `V1.0` 软件作品快照；后续正常 UI 修复、后端联调、登录、支付、OSS、SLS 或应用商店物料迭代不影响本次 `V1.0` 申请。
- 如果后续 App 名称、著作权人、核心软件形态或大版本发生重大变化，再评估补办新版本软著或办理变更。

## 官方入口

- 中国版权保护中心：https://www.ccopyright.com.cn/
- 进入著作权登记系统后选择：计算机软件著作权登记。

软著账号注册、企业实名认证、申请表确认、盖章、营业执照和经办授权等动作必须由公司或授权经办人完成，Codex 只准备可由仓库生成的技术材料和文档草稿。

## 本机已生成材料

材料目录：

- `D:/Desktop/农技千查软著材料`

当前文件：

- `软著申请信息草稿.txt`：填表口径、软件说明和功能点。
- `软著提交步骤简表.txt`：人工提交流程。
- `农技千查Android应用软件操作说明书.docx`：可作为软件文档材料草稿。
- `农技千查Android应用软件源代码摘录.docx`：按前后各 30 页、每页 50 行生成的源码摘录草稿。
- `源代码摘录-阅读版.txt`：只供人工核对，不建议直接当最终上传件。

生成原则：

- 源码摘录只取 Android 客户端 `app/src/main/kotlin/com/nongjiqianwen/*.kt`。
- 不取 Gradle release 签名配置、私钥、密码、本机 secret、模型 Key 或后端生产环境变量。
- 提交前仍应由经办人打开材料检查排版、申请信息、公司主体和盖章要求。
