一、网络访问

权限 / 能力：INTERNET。 调用场景：连接平台服务、短信验证码登录、上传图片、流式回答、会员权益、帮助与反馈、今日农情和检查更新。 调用时机：您使用在线功能或 App 需要同步服务状态时。 拒绝 / 关闭影响：没有网络访问时，在线问诊、登录、反馈和更新等功能无法正常使用。

二、网络状态

权限 / 能力：ACCESS_NETWORK_STATE。 调用场景：判断网络是否可用，在登录、上传、问诊、反馈和检查更新失败时给出提示。 调用时机：发起联网功能前或网络请求失败后。 拒绝 / 关闭影响：可能无法准确提示网络状态，但不影响您在系统层面管理网络连接。 边界：当前不申请 Wi-Fi 状态权限，不读取 Wi-Fi 列表。

三、安装更新包

权限 / 能力：REQUEST_INSTALL_PACKAGES。 调用场景：仅在您主动点击“立即更新”后，下载官方安装包并调起系统安装确认页。 调用时机：您主动检查更新并确认安装时。 拒绝 / 关闭影响：您可以继续使用当前版本，但无法通过 App 内自有更新入口安装新版本。 边界：App 不做静默安装，不后台自动安装，也不从非官方地址安装更新包。

四、后台待发送任务

权限 / 能力：WAKE_LOCK、RECEIVE_BOOT_COMPLETED、FOREGROUND_SERVICE 和应用私有广播保护权限。 调用场景：带图消息在您离开 App、进程被系统回收或设备重启后，对同一条待发送任务进行有限重试。 调用时机：仅存在您已主动发送但尚未完成的带图待发送任务时。 拒绝 / 关闭影响：系统可能延后或终止待发送任务，您需要重新打开 App 后重试。 边界：该能力不用于 App 外推送、广告通知、定位跟踪、读取通讯录或读取短信。

五、相机和照片

权限 / 能力：系统照片选择器、外部相机和 FileProvider 临时文件授权。 调用场景：您主动选择图片或拍照后用于问诊、帮助反馈和预览。 调用时机：您点击相机或照片入口时。 拒绝 / 关闭影响：无法使用图片问诊或图片反馈，但仍可文字问诊。 边界：当前不申请 App 相机权限，不申请相册 / 存储读写权限；照片入口只访问您本次主动选择的图片。拍照成功后，系统可能在相册 Pictures/农技千查 另存一份照片。

六、定位和未申请权限

权限 / 能力：ACCESS_COARSE_LOCATION。 调用场景：仅在问诊需要时申请粗略定位，用于生成省、市、区县等地区文本，辅助判断农时和地区背景。 调用时机：首次发送问诊或后续发送问诊前需要刷新地区文本时；不会在 App 首屏无场景弹出权限框。 拒绝 / 关闭影响：仍可文字或图片问诊，但回答会缺少您所在地区的农时背景，可能不够贴合当地情况。 边界：不向农技千查服务器上传经纬度，不保存轨迹，系统定位和地理编码能力按设备系统规则处理；不用于地图、广告、统计或推送。当前不申请相册 / 存储读写、录音、通讯录、短信、电话状态、Wi-Fi 状态或通知权限，也不做 App 外推送通知。
