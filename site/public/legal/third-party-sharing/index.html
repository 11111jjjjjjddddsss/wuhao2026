一、基本原则

我们不会出售您的个人信息。除依法依规、取得授权、实现服务必要委托处理、处理投诉争议或保护安全外，不会向无关第三方提供您的个人信息。我们会按 SDK、API 接口、云服务托管、系统能力等类型列明共享情况；如第三方服务新增或变更，我们会更新本清单。第三方具体处理规则以其公开隐私政策或与我们的服务约定为准。

二、智能分析和云服务

共享类型：服务端 API 接口 / 云服务委托处理。 服务提供方：阿里云计算有限公司及其关联服务平台。 服务类别：服务器侧云计算、数据存储和智能分析服务。 使用场景：农业问答、图片理解、记忆摘要、今日农情、历史恢复、故障排查和权益核对。 涉及信息：您主动提交的文字、图片、地区文本、必要上下文、AI 回复、账号标识、运行状态和安全诊断信息。 共享方式：由农技千查服务器在必要范围内调用服务接口，Android 端不直接连接第三方智能分析服务。 第三方处理规则：阿里云法律声明及隐私权政策，以及对应云产品公开规则或我们与其签署的服务约定。

三、云资源和存储

共享类型：云服务托管 / 服务端访问控制。 服务提供方：阿里云计算有限公司及其关联服务平台。 服务类别：云数据库、对象存储、缓存、日志和 HTTPS 基础设施。 使用场景：保存账号、会话、图片、会员权益、礼品卡、帮助反馈、检查更新、后台审计和安全诊断信息。 涉及信息：完成对应功能所需的账号标识、图片文件、反馈内容、订单 / 权益状态、版本和运行状态。 共享方式：由平台服务器托管和访问控制，按规则留存、清理或去标识化。 第三方处理规则：阿里云法律声明及隐私权政策，以及对应云产品公开规则或我们与其签署的服务约定。

四、手机号登录和短信服务

共享类型：服务端 API 接口。 服务提供方：阿里云计算有限公司及其短信服务平台。 服务类别：第三方短信验证码服务。 使用场景：发送登录验证码、校验账号登录请求、保障账号安全。 涉及信息：手机号、验证码发送状态、发送时间和必要的安全风控信息。 共享方式：由农技千查服务器调用短信服务接口发送验证码；验证码需要您手动输入，App 不读取短信内容、通讯录或通话记录。 第三方处理规则：阿里云法律声明及隐私权政策，以及短信服务公开规则或我们与其签署的服务约定。 拒绝影响：拒绝提供手机号将无法完成账号登录。

五、系统能力

共享类型：系统能力调用 / 用户主动授权。 服务提供方：Android 系统、系统照片选择器、您设备上的外部相机或系统安装确认页。 服务类别：Android 系统能力或您设备上的外部相机应用。 使用场景：选择照片、拍照、打开系统浏览器、下载后调起系统安装确认页。 涉及信息：您主动选择或拍摄的图片、系统安装确认状态和必要的本地文件授权。 共享方式：仅在您主动点击对应功能时由系统处理；这些能力不是 App 内嵌的广告、统计或推送 SDK。 处理规则：相关系统能力按设备系统规则处理；您可以不使用图片、拍照或更新安装功能，其他问诊和账号功能仍可继续使用。

六、支付服务

共享类型：支付宝 APP 支付 SDK / 支付服务接口。 服务提供方：支付宝（中国）网络技术有限公司及支付宝开放平台相关服务。 服务类别：支付渠道服务。 使用场景：您主动购买会员、加油包或其他付费服务时，调起支付宝完成支付。 涉及信息：订单号、商品 / 权益信息、支付金额、支付状态、退款和售后处理所需信息，以及支付宝 SDK 完成支付所需的设备和支付环境必要信息。 共享方式：由农技千查后端创建订单并生成支付参数，Android 端仅将支付参数交给支付宝 SDK 调起支付；权益发放以后端验签通知或订单状态核验为准，客户端同步结果不直接发放权益。 第三方处理规则：支付宝隐私权政策及支付宝开放平台公开规则。

七、当前未接入的第三方 SDK

当前版本不接入广告、地图、推送、统计、社交分享、通讯录、短信读取、电话状态或 Wi-Fi 状态类第三方 SDK。支付宝 SDK 仅在您主动购买付费服务时调用；基础开源组件和 Android 系统组件不作为独立第三方收集个人信息，相关数据处理仍以隐私政策和 App 内清单为准。
